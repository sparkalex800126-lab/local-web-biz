## ✅ 維護服務內容

### 1. 內容更新服務

| 項目 | 額度 | 說明 |
|------|------|------|
| 文字內容修改 | 每月最多 **2 次** | 營業時間、價格、服務說明等文字調整 |
| 圖片更換 | 每月最多 **2 次** | 產品照片、環境照片等更換（客戶提供新圖） |
| 節日 / 活動 Banner | 每年最多 **4 次** | 過年、中秋、週年慶等節慶 Banner 設計與更換 |
| 新增簡易內容 | 每月最多 **1 次** | 新增一則最新消息或簡短公告（方案 C 適用） |

> 💡 每次更新請一次彙整需求，透過 LINE 或 Email 傳送。我們會在 **3 個工作天內**完成更新。

### 2. 技術維護

| 項目 | 內容 |
|------|------|
| 網站正常運作監控 | 24/7 uptime 監控，異常時主動通知並處理 |
| SSL 憑證管理 | 確保 HTTPS 憑證持續有效、自動續約 |
| 主機維護 | 伺服器穩定運作、定期更新 |
| 網站備份 | 每月自動備份，保留最近 3 個月的備份 |
| Bug 修復 | 網站功能異常時免費修復（非人為因素造成） |
| 瀏覽器相容性 | 確保主流瀏覽器（Chrome、Safari、Edge）正常顯示 |

### 3. 基礎 SEO 維護

| 項目 | 內容 |
|------|------|
| Google Search Console | 持續監控網站索引狀態 |
| 基礎 SEO 檢查 | 每季檢查 meta 標籤、alt 標籤、sitemap 等基礎項目 |
| 網站速度監控 | 定期確認載入速度，優化明顯問題 |
| Google 商家資訊連動 | 確保網站與 Google 商家資訊一致（如客戶有 Google 商家） |

### 4. 客戶支援

| 項目 | 內容 |
|------|------|
| LINE / Email 諮詢 | 網站相關問題隨時可詢問 |
| 回覆時間 | 工作日 **24 小時內**回覆 |
| 緊急狀況 | 網站完全無法存取時，**4 小時內**開始處理 |

## 🔄 續約與不續約

### 續約

- 維護方案到期前 **30 天**，我們會主動聯繫您確認續約意願
- 續約價格維持 $4,800/年（如有調整會提前 60 天通知）
- 續約後服務不中斷，自動銜接

### 不續約

如果您決定不續約，請注意以下事項：

| 項目 | 處理方式 |
|------|----------|
| 網站原始碼 | 我們會提供完整原始碼給您 |
| 網站資料備份 | 提供最新一份完整備份 |
| 主機 | 維護到期後 **30 天**內需自行遷移至您自己的主機 |
| SSL 憑證 | 到期後需自行管理 |
| 域名 | 如由我方代管，將協助移轉至您指定的管理商 |
| 技術支援 | 到期後不再提供免費技術支援 |
| 內容更新 | 到期後如需更新，將以單次報價計費 |

> ⚠️ 維護到期後如未續約也未遷移，**30 天後**我方將停止提供主機服務，網站可能無法存取。請務必提前安排。
