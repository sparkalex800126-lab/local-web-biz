# PageReady TW — Outreach 訊息模板

> 最後更新：2026-03-21
> 所有模板使用繁體中文，語氣＝技術好的鄰居主動幫忙

## 變數說明

| 變數 | 說明 | 範例 |
|------|------|------|
| `{{store_name}}` | 店家名稱 | 小樹苗托嬰中心 |
| `{{owner_name}}` | 老闆/負責人稱呼 | 陳小姐 |
| `{{demo_url}}` | demo 網站連結 | https://demo.pagereadytw.cc/小樹苗 |
| `{{specific_observation}}` | 個人化觀察（Google 評論、社群、現況） | 看到你們 Google 評論都超好的 |
| `{{pain_point}}` | 對方可能的痛點 | 但搜「竹北托嬰」好像不太容易找到你們 |

## 一、LINE 訊息模板（首次接觸）

### 核心策略

demo 已經做好 → 對方零成本點開看 → CTA 是「看一下」不是「約時間」。

重點：**訊息要短到在 LINE 預覽就能看完重點。** 在地店老闆一天收幾十則 LINE，超過三行就跳過。

---

### 版本 A —「直球派」（最短，適合餐廳/美髮等忙碌業種）

```
{{owner_name}} 你好～

我是做網頁設計的，在竹北這邊。

最近幫 {{store_name}} 做了一個網站 demo，想說直接給你看比較快 👇

{{demo_url}}

免費的，純粹覺得你們店很讚想練個作品。
喜歡的話再聊，不喜歡也完全沒關係！
```

**設計邏輯：**
- 第一句建立身份＋在地感（「竹北這邊」）
- 第二句直接給連結，零廢話
- 第三句降低壓力（「免費」「練作品」「沒關係」）
- 沒有任何推銷字眼

---

### 版本 B —「觀察派」（有做功課，適合托嬰/月子等需要信任的業種）

```
{{owner_name}} 你好，

我住竹北，{{specific_observation}}，{{pain_point}}。

我自己會做網站，就順手幫 {{store_name}} 做了一個 demo：
{{demo_url}}

沒有要推銷啦～就是覺得你們值得有個好看的網站讓更多爸媽找到。
看看喜不喜歡，隨時跟我說 🙂
```

**個人化填寫範例：**
```
{{specific_observation}} = 看到你們 Google 評論幾乎都五顆星
{{pain_point}} = 不過搜「竹北托嬰中心」要滑很久才找得到
```

**設計邏輯：**
- 先展示你有做功課（觀察＋痛點）
- 「住竹北」強化鄰居感
- 「順手幫」暗示這不是大量推銷
- CTA 仍然是「看看」，不是「約見面」

---

### 版本 C —「共同連結派」（透過朋友/社群/活動認識的情況）

```
{{owner_name}} 你好！

我是 {{referral_name}} 的朋友，他跟我提過 {{store_name}}。

我在竹北做網頁設計，最近剛好在研究 {{industry}} 的網站，就順手幫你們做了一個 demo 👇

{{demo_url}}

完全免費，就當交個朋友。覺得可以的話再聊～
```

**設計邏輯：**
- 共同連結開場 → 信任直接拉高
- 「研究 {{industry}} 的網站」解釋為什麼會做 demo，合理化行為
- 「交個朋友」降壓

---

### 版本 D —「問題切入派」（沒有 demo 的情況，純探路用）

```
{{owner_name}} 你好～

我在竹北做網站設計的。想請教一下，{{store_name}} 現在新客人主要都怎麼找到你們的？

純粹好奇，因為我最近在研究竹北在地店家的網路行銷狀況。

如果你有興趣，我可以免費幫你做一個網站 demo 看看效果 🙂
```

**設計邏輯：**
- 用問題開場引發互動（Saraev 原則：問 what/why）
- 不直接推東西
- 備用方案：還沒做 demo 時先建立關係

## 三、使用指南

### 選哪個版本？

| 情境 | 建議版本 |
|------|---------|
| 餐廳、美髮、小吃店（老闆很忙） | LINE 版本 A |
| 托嬰中心、月子中心（需要信任） | LINE 版本 B |
| 有共同朋友介紹 | LINE 版本 C |
| 還沒做 demo，先探路 | LINE 版本 D |
| 只有 email | Email 版本 A 或 B |

### 個人化 Checklist（寄出前 2 分鐘檢查）

- [ ] Google 搜過店名，知道他們的基本資訊
- [ ] 看過 Google 評論，能提到一個具體的好評或特色
- [ ] 搜過「竹北 + 業種」，知道他們排第幾頁
- [ ] demo 連結能正常開啟、手機版排版正常
- [ ] 訊息讀起來不像群發（把店名拿掉後，這則訊息不能直接發給另一家店）

### ❌ 不要做的事

- 不要第一句就講價格
- 不要用「尊敬的老闆」「敬啟者」這種語氣
- 不要附加 PDF 提案或服務介紹
- 不要在第一則訊息就約見面
- 不要一次群發相同訊息給同條街的店家（會互相比對）
- 不要用簡體字
- 不要寫「AI 賦能」「數位轉型」「賦能升級」 — 講人話
