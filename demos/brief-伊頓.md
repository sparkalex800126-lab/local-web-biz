# Demo Brief: 伊頓國際托嬰中心

## 目標
做一個現代、溫馨的一頁式托嬰中心網站 demo，用來展示給業主看。

## 基本資訊
- 名稱：伊頓國際托嬰中心（Eton Kids International Daycare）
- 地址：新竹縣竹北市中華路2-7號
- 電話：03-553-7733
- 服務時間：週一至週五 07:00-19:30
- 收托對象：0-2 歲嬰幼兒
- 採預約參觀制

## 設計方向
- 溫暖、專業、現代感
- 配色：柔和的暖色調（米白、淺粉、柔綠、木質色）
- 字體：圓體或柔和的 sans-serif
- 整體感受：讓家長一看就覺得「這裡安全、溫暖、專業」

## 頁面結構（一頁式）
1. Hero Section — 大圖 + 中心名稱 + 一句 slogan + CTA「預約參觀」
2. 關於我們 — 簡短的理念介紹
3. 服務特色 — 3-4 個特色卡片（師生比、環境安全、課程、即時監控等）
4. 環境展示 — 照片 gallery（用 placeholder）
5. 家長見證 — 2-3 則 testimonial（placeholder）
6. 聯絡資訊 — 地址、電話、地圖嵌入、預約表單

## 技術要求
- 純 HTML + CSS + 少量 JS
- 響應式設計（手機優先，家長都用手機看）
- 不需要後端
- 圖片用 placeholder（unsplash 或 SVG placeholder）
- 可以直接部署到 GitHub Pages

## 重要
- 繁體中文內容
- 所有文案要像真實網站，不要用 Lorem ipsum
- 照片用 Unsplash 的嬰幼兒/托嬰相關免費圖片 URL
